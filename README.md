# WoW Price Checker

A World of Warcraft Auction House price checker using the official Blizzard API.

## Features

- Fetch real-time auction house data
- Query character equipment and profile information
- Track item prices across realms

## Technologies

- Blizzard Battle.net API
- OAuth 2.0 Authentication

## Setup

1. Create a Blizzard API application at https://develop.battle.net/
2. Get your Client ID and Client Secret
3. Configure your credentials (see `.env.example`)

## API Endpoints

### Character Profile
- Equipment data
- Collections (mounts, pets, toys)
- Professions and statistics

### Auction House
- Real-time auction listings
- Historical price data
